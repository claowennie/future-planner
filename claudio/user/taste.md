# 我的听歌口味（蒸馏自 QQ 歌单「I LIKE」，554 首，2026-05-31）

> 给 Claudio 选歌时参考。这是从一整张歌单提炼出的画像，不是硬清单——
> 抓气质、别被字面困住。锚点歌见 playlists.json。

## 一句话
英文 indie / bedroom-pop 是主轴，华语流行与说唱是情怀底色，再缀上动漫和
电影配乐的电影感。整体偏**柔、内省、夜晚感**，但也吃得下明快流行和一点 phonk 提神。

## 语言占比（大致）
- 英文及其他 ~70%
- 华语 ~20%
- 韩语 ~6%（K-pop）
- 日语 ~4%（多为动漫相关）

  ROOK1E、vietra、Lyn Lapid、The Walters、Ghostly Kisses —— 这一脉是我的舒适区。
- **西方流行**：Taylor Swift、Lana Del Rey、Coldplay、Harry Styles、SZA、The Weeknd、
  Melanie Martinez、Conan Gray、Gracie Abrams、Chappell Roan、Lady Gaga。
- **爵士 / 暖调**：Laufey、Bruno Major。
- **华语流行**：周杰伦、林俊杰、王力宏、胡彦斌、邓紫棋、方大同、李荣浩、毛不易、
  汪苏泷、蔡健雅 —— 偏经典与抒情，带怀旧滤镜。
- **华语说唱 / R&B**：法老、王以太、艾热、丁世光。
- **K-pop**：NewJeans、BABYMONSTER、BLACKPINK、EXO。
- **日系 / 动漫**：RADWIMPS（《你的名字。》）、宇多田光（EVA）、Carole & Tuesday 群像、和ぬか。
- **88rising 一脉**：Joji、Rich Brian、NIKI、Higher Brothers。

## 电影 / 游戏 / 动漫配乐（约 9%，很在意）
Hans Zimmer《星际穿越》、Alexandre Desplat《水形物语》、Evan Call《紫罗兰永恒花园》、
Andrew Prahlow《Outer Wilds》、Daniel Pemberton《Project Hail Mary》、
古典钢琴（Gymnopédie、The Piano Guys、大提琴）。适合做「专注 / 放空」的底。

## 情绪倾向
- 默认偏**安静、内省、略忧郁**的歌——深夜、独处、想事情时的状态。
- 喜欢**真声学 / 弹唱 / 翻唱**质感（Jada Facer 那类不插电翻唱常出现）。
- 需要打起精神时，会要**明快流行**或**phonk / 鼓点**。
- 对**OST / 纯音乐**接受度高，可作无人声的工作背景。

## 给 Claudio 的提示
- 串歌时可在「英文 indie ↔ 华语情怀」之间自然切换，这是我的两个家。
- 夜晚 / 安静时段优先柔的、卧室流行那挂；白天 / 通勤可以更明快。
- 点华语经典（周杰伦、林俊杰那批）容易戳中我，但别每次都靠它。
